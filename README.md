# TestCafé Boilerplate

TestCafé end-to-end test automation boilerplate, Using ES Modules.

## Getting Started

    	1. git clone git@github.com:e2e-boilerplate/testcafe-es-modules.git
    	2. cd testcafe-es-modules
    	3. npm install
    	4. npm run test
