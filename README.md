# TestCafé Boilerplate

TestCafé end-to-end test automation boilerplate, Using ES Modules, Chai and Expect.

## Getting Started

    	1. git clone git@github.com:e2e-boilerplate/testcafe-es-modules-chai-expect.git
    	2. cd testcafe-es-modules-chai-expect
    	3. npm install
    	4. npm run test
